# Tecnical Information 

## Components

1. **Frontend**: 
    - Built with [React Native](https://reactnative.dev/). 
    - Uses [Expo](https://expo.io/) for development and deployment.

    - [Expo sqlite](https://docs.expo.dev/versions/latest/sdk/sqlite/) for local storage.
